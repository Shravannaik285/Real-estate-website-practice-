
                        🏡 Real Estate Website (Frontend Practice Project)
                        


## 🛠 Technologies
    -React.js
    -Tailwind CSS
    -JavaScript (ES6)
    -HTML5
    -CSS3
    -Vite
    -Netlify (Deployment)
    -Git & GitHub
## ✨ Features
    -Responsive design (Mobile, Tablet, Laptop)
    -Clean and modern UI layout
    -Hero section with property search inputs
    -Featured properties section
    -Property categories section
    -About page with company details
    -Contact page with form and details
    -Reusable React components
    -Smooth navigation between sections/pages
## ⌨️ Keyboard Shortcuts (During Development)
    -Ctrl + S → Save file
    -Ctrl + / → Comment code
    -Ctrl + C → Stop server
    -Ctrl + `` → Open terminal
    -npm run dev → Run development server
## ⚙️ The Process
    -Planned the structure of the real estate website
    -Designed layout for Home, Properties, About, and Contact pages
    -Built UI components using React
    -Styled components using Tailwind CSS
    -Made the website fully responsive for different screen sizes
    -Fixed bugs and improved layout step-by-step
    -Deployed the project using Netlify
    -Uploaded code to GitHub
